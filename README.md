# ⚡ DaySnap – Plan Your Day in a Snap

A clean, minimal, and lightning-fast daily productivity app built with Streamlit.

DaySnap helps you focus only on **today** — no clutter, no overwhelm, just simple task planning that actually gets used.

---

## 🚀 Live Concept

> “What’s the simplest way to plan my day right now?”

DaySnap is designed for:

* Students
* Freelancers
* Anyone who hates complex productivity apps

---

## ✨ Features

### 📝 Today Tasks

* Add tasks instantly (press Enter or click)
* Mark tasks as complete
* Delete tasks easily
* No categories, no tags, no distractions

---

### 🔁 Auto Reset System

* Tasks reset automatically every new day
* No manual cleanup needed
* Removes guilt from unfinished tasks

---

### 🌿 Daily Routine Checklist

* Predefined habits:

  * 🌅 Wake up early
  * 🏃 Exercise
  * 📚 Study
  * 💧 Drink water
* Track daily consistency

---

### 🔥 Streak System

* Tracks consecutive productive days
* Displays:

  * `🔥 X Day Streak`
* Resets if you skip a day

---

### 📊 Progress Tracking

* Shows task completion:

  * `X/Y tasks completed`
* Visual progress bar
* Motivational messages:

  * “Good start 🚀”
  * “Almost done 🔥”

---

### 🌙 Clean UI / UX

* Minimal and distraction-free
* Mobile-friendly design
* Light & Dark mode toggle
* Smooth animations and modern styling

---

  * Routine checklist

  * Last active date
  * Theme preference
